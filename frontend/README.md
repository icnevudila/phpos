# DentQL

DentQL is a modern, product-led dental clinic operating system. 

## Phase
The project has completed the **UI/UX & Interactive Prototype Phase** and is now transitioning into the **Production Backend Phase** utilizing Supabase (PostgreSQL).

## Tech Stack
- React 18 (TypeScript)
- Vite
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations)
- React Router (Routing)
- Supabase (Backend/Auth/Database) - *In Progress*

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your Supabase credentials.
4. Run `npm run dev` to start the local development server.
